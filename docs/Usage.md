# Usage Guide

This guide walks through daily tasks in Kue.

## Keybindings

The following shortcuts are available throughout the application:

| Key | Action |
|-----|--------|
| `q`, `esc`, `ctrl+c` | Quit / Return |
| `↑`, `k` | Move up |
| `↓`, `j` | Move down |
| `→`, `l` | Next / Right |
| `←`, `h` | Previous / Left |
| `ctrl + d` | Delete queue / message |
| `ctrl + n` | Create queue / message |
| `?` | Toggle help |
| `enter` | View selected item |
| `space` | Select item |
| `/` | Filter |
